# 🎯 Preview Local - Guia Rápido

iADV rodando **100% localmente** com dados fictícios.

## ⚡ Rodar Agora

```bash
cd /Volumes/Dados/APP-vendas-projetos
npm install
npm run dev
```

**Acesse:** http://localhost:3000

## 📊 O que você vai ver

### 1. **Inbox** (Página Principal)
- 4 itens de exemplo (intimações, publicações, andamentos)
- Filtros por tipo e status
- Estatísticas no topo
- Quick actions (visual apenas)

### 2. **Prazos**
- 5 prazos (1 atrasado, 1 para hoje, 3 futuros)
- Widgets "5 minutos por dia"
- Tabela completa com processo, cliente, prioridade

### 3. **Processos**
- 3 processos com CNJ real formatado
- Vinculados a clientes
- Status e tribunal

### 4. **Clientes**
- 3 clientes (2 PF, 1 PJ)
- CPF/CNPJ formatados
- Telefone e email

### 5. **Outras Páginas**
- Documentos (página base)
- Configurações (formulários)
- Ajuda (docs)
